# Awesome ChatGPT Prompts 的历史

## 起点：2022 年 11 月

当 ChatGPT 于 2022 年 11 月首次发布时，AI 的世界在一夜之间发生了改变。曾经只属于研究人员和开发者的领域，突然向所有人敞开。Fatih Kadir Akın 是被这项新技术深深吸引的人之一，他是一位开发者，在 ChatGPT 的能力中看到了非凡之处。

> "当 ChatGPT 首次发布时，我立刻被它的能力所吸引。我以各种方式尝试这个工具，每次的结果都让我惊叹不已。"

那些早期的日子充满了实验和发现。世界各地的用户都在寻找与 ChatGPT 互动的创意方式，分享他们的发现，相互学习。正是在这种兴奋和探索的氛围中，"Awesome ChatGPT Prompts"的想法应运而生。

## 开创一切的仓库

2022 年 12 月，ChatGPT 发布仅几周后，[Awesome ChatGPT Prompts](https://github.com/f/prompts.chat) 仓库在 GitHub 上创建。这个概念简单但强大：一个精心策划的有效提示词集合，任何人都可以使用和贡献。

这个仓库迅速获得关注，成为全球 ChatGPT 用户的首选资源。最初只是一个有用提示词的个人收藏，后来发展成为一个社区驱动的项目，收到了来自全球各地开发者、作家、教育者和爱好者的贡献。

## 第一本书：《ChatGPT 提示词的艺术》

仓库的成功催生了《ChatGPT 提示词的艺术：编写清晰有效提示词的指南》——一本于 2023 年初在 Gumroad 上出版的综合指南。

这本书捕捉了提示词工程的早期智慧，涵盖：

- 理解 ChatGPT 的工作原理
- 与 AI 清晰沟通的原则
- 著名的"扮演..."技巧
- 逐步编写有效提示词
- 常见错误及如何避免
- 故障排除技巧

**这本书引起轰动**，在 Gumroad 上实现了超过 **10 万次下载**。它在社交媒体上广泛传播，在学术论文中被引用，被社区成员翻译成多种语言。令人意外的是，它还获得了知名人士的认可——甚至连 OpenAI 的联合创始人兼总裁 [Greg Brockman](https://x.com/gdb/status/1602072566671110144) 也认可了这个项目。

## 奠定领域基础的早期见解

在那些形成期的几个月里，出现了几个关键见解，后来成为提示词工程的基础：

### 1. 具体性很重要

> "我学到了使用具体且相关的语言的重要性，以确保 ChatGPT 理解我的提示词并能够生成适当的回复。"

早期的实验者发现，模糊的提示词会导致模糊的回答。提示词越具体、越详细，输出就越有用。

### 2. 目的和焦点

> "我发现了为对话定义明确目的和焦点的价值，而不是使用开放式或过于宽泛的提示词。"

这一见解成为后来几年发展的结构化提示技术的基础。

### 3. "扮演..."的革命

社区中出现的最具影响力的技术之一是"扮演..."模式。通过指示 ChatGPT 扮演特定角色或人格，用户可以显著提高回复的质量和相关性。

```
我想让你扮演一个 JavaScript 控制台。我会输入命令，你会回复
JavaScript 控制台应该显示的内容。我只希望你在一个唯一的代码块
中回复终端输出，不要有其他内容。
```

这个简单的技术开启了无数可能性，至今仍是最广泛使用的提示策略之一。

## prompts.chat 的演变

### 2022：起步

项目开始时只是一个简单的 GitHub 仓库，一个在 GitHub Pages 上渲染为 HTML 的 README 文件。它很简陋但功能完备——这证明了伟大的想法不需要复杂的实现。

**技术栈**：HTML、CSS、GitHub Pages

### 2024：界面更新

随着社区的壮大，对更好用户体验的需求也在增长。网站进行了重大的界面更新，借助 Cursor 和 Claude Sonnet 3.5 等 AI 编程助手完成。

### 2025：当前平台

如今，prompts.chat 已发展成为一个功能完善的平台，采用：

- **Next.js** 作为 Web 框架
- **Vercel** 进行托管
- 使用 Windsurf 和 Claude 进行 **AI 辅助开发**

平台现在具有用户账户、收藏、搜索、分类、标签，以及一个蓬勃发展的提示词工程师社区。

### 原生应用

项目扩展到了网页之外，推出了用 SwiftUI 构建的原生 iOS 应用，将提示词库带给移动用户。

Awesome ChatGPT Prompts 项目对人们与 AI 互动的方式产生了深远影响：

## 理念：开放与免费

从一开始，该项目就致力于开放。在 CC0 1.0 通用（公共领域贡献）许可下，所有提示词和内容都可以自由使用、修改和分享，没有任何限制。

这一理念促成了：

- 多语言翻译
- 整合到其他工具和平台
- 学术使用和研究
- 商业应用

目标始终是让有效的 AI 沟通技术民主化——确保每个人，无论技术背景如何，都能从这些工具中受益。

## 三年之后

ChatGPT 发布三年后，提示词工程领域已经显著成熟。最初只是非正式的实验，现已发展成为一个公认的学科，拥有既定的模式、最佳实践和活跃的研究社区。

Awesome ChatGPT Prompts 项目与这个领域一起成长，从一个简单的提示词列表演变为一个用于发现、分享和学习 AI 提示词的综合平台。

这本书代表了下一次演变——三年社区智慧的提炼，为当今和未来的 AI 格局进行了更新。

## 展望未来

从最初的仓库到这本综合指南的历程，反映了 AI 的快速发展以及我们对如何有效使用它的理解的深化。随着 AI 能力的不断进步，与这些系统沟通的技术也将不断发展。

那些早期发现的原则——清晰、具体、目的明确，以及角色扮演的力量——至今仍然适用。但新技术不断涌现：思维链提示、少样本学习、多模态交互等等。

Awesome ChatGPT Prompts 的故事归根结底是一个关于社区的故事——关于世界各地成千上万的人分享他们的发现、互相帮助学习，共同推进我们对如何与 AI 协作的理解。

这种开放合作和共同学习的精神，正是这本书希望延续的。
